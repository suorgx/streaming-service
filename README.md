# Streaming service

## About project:

A streaming service like Netflix that offers authentication, the ability to watch content, and the option to add your favorite movies to a personal collection

## Stack:

- Next.js
- Typescript
- Tailwind
- Axios
- Prisma
- MangoDB
- bcrypt
